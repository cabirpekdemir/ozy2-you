🔧 Step 2 — Install (macOS)

Installation in 3 clicks

1

Open the DMG file

Double-click OZY2-v2.2.0-mac.dmg → A window opens showing the OZY2 app icon and your Applications folder

2

Drag to Applications

Drag the OZY2 icon → drop it onto the Applications folder shortcut in the same window

3

Launch & Start Onboarding

Open Applications → double-click OZY2 → OZY2 appears in your menu bar (top-right ⚙️) → click it → "Open Dashboard" → browser opens at localhost:8080

⚠️ If macOS says "unidentified developer": Right-click OZY2 → Open → Open anyway (one-time only)

🎉 Step 3 — Onboarding

First-run setup wizard

1

AI Provider Setup

Choose your AI: Gemini (recommended — free tier available), OpenAI, Anthropic, or Ollama (fully local, no internet). Paste your API key.

2

Name Your AI & Pick an Avatar

Give your assistant a name (default: OZY) and choose from 12 emoji avatars. This is YOUR personal assistant.

3

Tell it about yourself

Name, age, gender, blood type, interests, dietary preferences. All stored locally. Your AI uses this to personalize responses.

🔑 API Keys — How to Get Them

Choose your AI engine

✨ Google Gemini

Recommended for beginners

1. Go to aistudio.google.com
2. Sign in with Google
3. Click "Get API key" → "Create API key"
4. Copy the key → paste in OZY2 Settings

FREE TIER — 15 req/min
aistudio.google.com/app/apikey
🦙 Ollama (Local)

100% Offline — No API key needed

1. Download from ollama.com
2. Run: ollama pull llama3.2
3. In OZY2 Settings → Provider: Ollama
4. No API key required!

COMPLETELY FREE
ollama.com
🤖 OpenAI

GPT-4o / GPT-4

1. Go to platform.openai.com
2. Sign up → Billing → Add credits
3. API Keys → Create new key
4. Copy → paste in OZY2 Settings

PAID — ~$0.01/1K tokens
platform.openai.com/api-keys
🧠 Anthropic Claude

Claude 3.5 Sonnet

1. Go to console.anthropic.com
2. Sign up → API Keys
3. Create key → Copy
4. Paste in OZY2 Settings

PAID — ~$0.003/1K tokens
console.anthropic.com

💡 Tip: Start with Google Gemini — free tier is generous enough for daily use
