## Nasıl Eklenir?

1. **github.com/cabirpekdemir/ozy2-you** adresine git
2. Repo adının yanındaki ⚙️ ikonuna tıkla (sağ üstte)
3. "Topics" alanına yukarıdaki kelimeleri tek tek yaz, Enter'a bas
4. **Save changes**

> ⚠️ Maksimum 20 topic önerilir (fazlası spam görünür). Öncelikli 20'yi seç.
