**Title:**
```
I built a local-first AI assistant with 38 tool integrations – no account, no cloud, runs on Mac/Win/Linux
```

**Body:**
```
Hey r/selfhosted!

I've been working on OZY2 — a personal AI assistant that runs entirely on your machine. No account, no subscription, no data sent anywhere except to the AI provider you choose.

**How it works:**
- You paste your own API key (Gemini, OpenAI, Claude, or Ollama for fully offline)
- OZY2 starts a local FastAPI server on localhost:8081
- A browser UI opens with real tool use — not just chat

**38 tools across 3 tiers:**
- 🧑 YOU (free): Weather, Web Search, News, Currency, Notes, Reminders, Tasks, Word/Excel, Memory, Calendar
- ⚡ PRO (coming soon): Gmail, Google Drive, GitHub, Notion, Obsidian, Trello, Stocks, Daily Briefing
- 🌐 SOCIAL (coming soon): Discord, Spotify, YouTube, WhatsApp, Telegram, Debate Mode

**Tech stack:** Python + FastAPI backend, vanilla JS frontend, SQLite for local data, PyInstaller for packaging

The whole thing is open source under MIT.

GitHub: https://github.com/cabirpekdemir/ozy2
```
