Why OZY2
Built differently

Every design decision starts with one question: is this good for the user, or for us?

🔒
100% Local
Everything runs on your machine. Your conversations, API keys, and files never touch our servers — because we don't have any.
🤖
Any AI Provider
Bring your own key. OZY2 works with Gemini, GPT-4o, Claude, and Ollama. Switch providers anytime without losing your data.
⚡
38 Integrated Tools
From weather to GitHub to Spotify — OZY2 connects to the tools you already use. No copy-pasting between apps.
🖥️
All 3 Platforms
Native installers for macOS, Windows and Linux. Double-click to install, double-click to launch. No terminal required.
🆓
Open Source & Free
The full source code is on GitHub. Audit it, fork it, self-host it. No subscription, no account, no lock-in.
🧠
Persistent Memory
OZY2 remembers facts about you across conversations. Tell it once — it knows forever. Stored locally in SQLite.
Getting Started
Up in 60 seconds
1
Download
Grab the installer for your OS from GitHub Releases. One file, no dependencies.
2
Install
macOS: drag to Applications. Windows: run the .exe. Linux: run install.sh.
3
Add your key
Paste your AI provider API key in the setup wizard. Gemini has a free tier — no credit card needed.
4
Start talking
OZY2 opens in your browser. Ask it anything. Connect the integrations you need.
